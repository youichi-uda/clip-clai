## Description

clip-clai is a command-line tool that lets you read, edit, and automate Clip Studio Paint (.clip) files -- powered by AI integration with Claude Code.

What you can do:

[Free - no license required]
- View file info and layer structure
- Inspect internal database
- Export thumbnails
- Export a single layer per session

[Pro - with license]
- Export all layers as PNG
- Rename layers in bulk
- Edit layer attributes (opacity, visibility, blend mode, etc.)
- Batch processing across multiple files
- Full Claude Code Skill integration for AI-assisted workflows

Requirements:
- Node.js 18+
- npm
- (Optional) Claude Code for AI Skill integration

Installation:
  npm install -g clip-clai
  clip-clai activate <your-license-key>
